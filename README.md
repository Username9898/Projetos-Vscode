## 📁 Projetos Disponíveis

### 1. Criador de Páginas
**Descrição:** Gerador de landing pages com IA usando a API Groq.

**Funcionalidades:**
- Geração automática de páginas HTML/CSS profissionais
- Interface intuitiva e moderna
- Design responsivo e criativo
- Uso de emojis e animações CSS
- Visualização em tempo real

**Como usar:**
1. Abra o arquivo `Criador de Páginas/index.html` no navegador
2. Descreva seu negócio no campo de texto
3. Clique em "Gerar" para criar sua página
4. O código HTML será exibido ao lado com preview

**Tecnologias:** HTML, CSS, JavaScript, API Groq

### 2. A HP 12C - Calculadora Financeira Revolucionária
**Descrição:** Calculadora financeira completa inspirada na HP 12C, com funções básicas e avançadas.

**Funcionalidades Básicas:**
- Soma, Subtração, Multiplicação e Divisão

**Funções Financeiras Clássicas:**
- Juros Simples (J = P * i * t)
- Juros Compostos (M = P * (1+i)^n)
- Valor Presente (VP)
- Valor Futuro (VF)
- PMT - Parcelamento
- TIR - Taxa Interna de Retorno
- VPL - Valor Presente Líquido

**Funções Revolucionárias (Novas):**
- Simulador de Aposentadoria
- Calculadora de Investimentos em Ações
- Conversor de Moedas (Offline)
- Calculadora de Hipoteca
- Simulador de Negócios
- Calculadora de Juros de Cartão de Crédito
- Planejador Financeiro Pessoal
- Calculadora de Empréstimos

**Características:**
- ✅ 100% Offline - não precisa de internet
- ✅ Interface colorida e intuitiva
- ✅ Cálculos rápidos em batch
- ✅ Funções exclusivas não encontradas em calculadoras comuns

**Como usar:**
1. Execute o arquivo `A HP 12C/calculadora.bat`
2. Escolha a função desejada no menu
3. Insira os valores solicitados
4. Visualize o resultado

**Tecnologias:** Windows Batch Script (.bat)

## 🚀 Como Executar os Projetos

### Projetos Web (HTML/JS)
A maioria dos projetos web pode ser aberta diretamente no navegador:
```bash
# Navegue até a pasta do projeto
cd "Nome do Projeto"

# Abra o index.html no navegador
start index.html
```

### Projetos com Backend (Node.js)
```bash
# Navegue até a pasta do projeto
cd "Nome do Projeto"

# Instale as dependências
npm install

# Configure o arquivo .env com suas credenciais
cp .env.example .env

# Inicie o servidor
npm start
```

### Projetos com Docker
```bash
# Navegue até a pasta do projeto
cd "Nome do Projeto"

# Inicie com Docker Compose
docker-compose up
```

## 📋 Requisitos Gerais

- **Node.js** (para projetos backend)
- **Docker e Docker Compose** (para projetos com banco de dados)
- **Navegador moderno** (para projetos frontend)
- **Windows** (para calculadora HP 12C)

## 🔧 Configuração

1. Clone o repositório:
```bash
git clone https://github.com/Username9898/Projetos-Vscode.git
```

2. Instale as dependências em cada projeto:
```bash
cd "Nome do Projeto"
npm install
```

3. Configure as variáveis de ambiente:
```bash
cp .env.example .env
# Edite o arquivo .env com suas credenciais
```

## 📝 Notas Importantes

- **Segurança:** Nunca comite arquivos `.env` com credenciais reais
- **API Keys:** Substitua as chaves de API nos arquivos `.env.example` pelas suas próprias
- **Offline:** A calculadora HP 12C funciona completamente offline
- **IA:** Alguns projetos requerem chaves de API de serviços de IA (Groq, OpenAI)

## 🛠️ Tecnologias Utilizadas

- **Backend:** Node.js, Express
- **Banco de Dados:** PostgreSQL, MongoDB
