# 📦 StockPro - Gestão de Estoque Inteligente

Sistema completo de gestão de estoque com integração de Inteligência Artificial via Groq.

## 🛠️ Stack Tecnológico

- **Frontend:** React.js + Vite
- **Backend:** Node.js + Express
- **Banco de Dados:** PostgreSQL
- **Autenticação:** JWT
- **Inteligência Artificial:** Groq AI (Modelo: `llama-3.3-70b-versatile`)
- **Infraestrutura:** Docker & Docker Compose

## 🚀 Como rodar

```bash
git clone https://github.com/Username9898/StockPro.git
cd StockPro
cp .env.example .env
docker-compose up --build
```

- **Frontend:** http://localhost:5173
- **Backend API:** http://localhost:3000
